```
docs/

```

## 主文档位置

项目的核心文档位于根目录：

| 文件名 | 说明 |
|--------|------|
| `NX12MKE.md` | **主文档**：NX12 MKE板类零件自动化加工规则库手册（v2.0） |
| `AGENTS.md` | **行为准则**：AI助手开发规范与核心经验 |

## 文档维护原则

### 核心文档
- **NX12MKE.md**：包含所有规则定义、语法规范、工艺常量、测试方法等核心内容
- **AGENTS.md**：包含XML架构经验、MKE规则边界控制、大文件处理等开发经验

### 辅助文档
- 仅在必要时创建新的辅助文档
- 避免重复内容，优先更新主文档
- 历史报告、对比分析等过时内容及时清理

### 清理策略
- 删除已整合到主文档的临时报告
- 删除v1.3→v2.0等历史对比报告
- 保持项目简洁，只保留有价值的文档
